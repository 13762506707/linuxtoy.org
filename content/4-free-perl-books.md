[Perl](http://perl.org) 是一门通用的编程语言，由 Larry Wall 创建于 1987 年，被应用于系统管理、Web 编程、数据库处理、XML 处理等诸多方面。本文向你推荐 4 本免费的 Perl 编程书籍，让你对 Perl 语言达到登堂入室。

<!-- PELICAN_END_SUMMARY -->

### Beginning Perl by Simon Cozens, Peter Wainwright

![Beginning Perl](http://i.linuxtoy.org/images/2010/02/bp.jpg)

这是一本适合 Perl 新手的入门书籍，涵盖 Perl 语法基础、使用文件/数据库、开发 CGI Web 应用、面向对象编程等内容。此书可[在线阅读，也有 PDF](http://www.perl.org/books/beginning-perl/) 可下载。

### Higher-Order Perl by Mark Jason Dominus

![Higher-Order Perl](http://i.linuxtoy.org/images/2010/02/hop.jpg)

本书由 Perl 社区中的名人 Mark Jason Dominus 所著，是对库函数很好的补充，适合高级 Perl 程序员阅读。《Higher-Order Perl》提供有 [PDF 和 MOD 源](http://hop.perl.plover.com/book/)下载。

### Embedding Perl in HTML with Mason by Dave Rolsky, Ken Williams

![Embedding Perl in HTML with Mason](http://i.linuxtoy.org/images/2010/02/epm.jpg)

此书向你展示如何使用 Mason 来创建大型、复杂、动态的网站。其[在线版](http://www.masonbook.com/book/)可在这里找到。

### Practical mod_perl by Stas Bekman, Eric Cholet

![Practical mod_perl](http://i.linuxtoy.org/images/2010/02/pmp.gif)

该书为 Perl Web 开发者和 Web 管理员所写，对 mod\_perl 进行了深入而全面的探讨。《Practical mod\_perl》有 [HTML 和 PDF 版本](http://www.modperlbook.org/)可用。
