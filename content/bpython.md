[bpython](http://bpython-interpreter.org/) 是一个相当不错的 Python shell。跟 Python 默认自带的解释器相比，bpython 包括语法高亮显示、自动补完、函数参数列表提示、Rewind 代码、发送代码到 Pastebin、保存代码到文件等功能。
