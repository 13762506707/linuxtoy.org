[CoVim][c] 是一个新近出现的 Vim 插件。利用该插件，我们可以在 Vim
这个经典的文本编辑器中进行远程、实时地协作编辑。感兴趣的同学大可一试！

![CoVim](http://lt-file.b0.upaiyun.com/files/2013/05/covim.gif)

CoVim 托管在 GitHub 上，其[安装及说明][i]可从其 README.md 了解。
