Title: DiskInternals Linux Reader: 从 Windows 访问 Ext2/Ext3 文件系统
Date: 2007-12-16 08:30
Author: toy
Category: Apps
Slug: diskinternals-linux-reader

免费软件 [DiskInternals Linux
Reader](http://www.diskinternals.com/linux-reader/) 让你从 Windows
直接访问 Ext2/Ext3 Linux 文件系统，这应当为那些同时使用 Linux 和 Windows
的朋友带来方便。值得提醒的是，这种访问是只能读取而不能写入，因此具有一定的局限。

[![Linux
Reader](http://i.linuxtoy.org/i/2007/12/linux-reader-thumb.png)](http://i.linuxtoy.org/i/2007/12/linux-reader.png)

DiskInternals Linux Reader
很容易使用，其界面与资源管理器类似。启动该程序后，它会自动发现硬盘上的
Ext2/Ext3 Linux
分区，双击即可进入。如果需要提取某些文件，那么可以在选择后，点击工具栏中的
Save 按钮。

遗憾的是，DiskInternals Linux Reader 对中文支持不好。我在测试时发现，在
Linux 分区中的中文目录和文件皆显示为乱码。

DiskInternals Linux Reader 可用于 Windows 全系列版本，包括 Windows 95,
98, ME, NT, 2000, XP, 2003 Server, Vista。你可从这里下载 [DiskInternals
Linux Reader 的最新版
1.1](http://www.diskinternals.com/download/Linux_Reader.exe)。
