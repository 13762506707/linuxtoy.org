Title: Gentoo Linux 10.0 Test 放出
Date: 2009-09-28 15:26
Author: toy
Category: Distros
Tags: Gentoo
Slug: gentoo-linux-100-test-is-out

为庆祝 Gentoo Linux 诞生 10 周年，Gentoo Linux 开发小组计划发行 Gentoo
Linux  
10.0 Live DVD 特别版。目前，该 Live DVD 的 Test ISO  
映像已经放出，欢迎各位测试。

Gentoo Linux 10.0 Test 支持 x86 及 x86\\\_64 架构，主要包含下列组件：

* Linux Kernel 2.6.30  
* glibc 2.9  
* GCC 4.3.2  
* KDE 4.3.1 (默认桌面环境)  
* GNOME 2.24  
* Xfce 4.6.1

参阅[发布公告](http://www.gentoo.org/news/20090927-livedvd-bugs-announce.xml)可以了解进一步的信息。Gentoo  
Linux 10.0 Test
可从[这里下载](http://mirrors.sfo1.gni.com/Gentoo/ten/)。
