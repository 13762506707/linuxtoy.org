《[Learn Linux The Hard Way][l]》（笨办法学 Linux）根据 Zed A. Shaw 的
[Learn Code the Hard Way][c] 课程风格编写而成。它不仅提供基于 Web
的交互式环境，而且包括 30 个实用的练习，相信对 Linux
新手的学习必将带来帮助。
