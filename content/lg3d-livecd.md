Title: LG3D LiveCD：基于 Java 的三维桌面系统
Date: 2007-07-18 08:07
Author: MDZ
Category: Distros
Slug: lg3d-livecd

[G3D LiveCD](https://lg3d-livecd.dev.java.net/Web-Site/Welcome.html)
是可启动的 Linux 光盘，基于 Java 技术，旨在以 3D
窗口和可视化性能为桌面及应用软件提供更丰富的用户体验。我们可以到其官方网站下载光盘镜像，然后刻录成
CD 或在虚拟机中使用。其特点是能将普通的窗口程序三维化（比如我们可以用 3D
化过的 Firefox 浏览网页），当然，它不是 XGL。

[![LG3D](http://i.linuxtoy.org/i/2007/07/lg3d_s.jpg)](http://i.linuxtoy.org/i/2007/07/lg3d.jpg)

想要体验这个令人惊奇的 3D 桌面？那么去[下载该 LG3D
LiveCD](https://lg3d-livecd.dev.java.net/Web-Site/Download.html) 吧。
