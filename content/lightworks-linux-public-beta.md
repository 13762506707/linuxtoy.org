Title: Lightworks Linux 公开 Beta 版
Date: 2013-05-01 09:16
Author: lovenemesis
Category: Movie Player
Tags: lightworks
Slug: lightworks-linux-public-beta

经过 2 年漫长的等待，专业级别的非线编软件 Lightworks 终于发布了适用于
Linux 平台的 Beta 版本。

大名鼎鼎的 Lightworks
具备的功能在此不再赘述，请前往[官网](http://www.lwks.com/index.php?option=com_content&view=article&id=57&Itemid=182)或问各大搜索引擎。

关于此 Linux 平台下的 Beta 版本，有以下几点特别注意：

-   仅支持 64 位系统。
-   目前官方仅支持 Ubuntu 和 Linux Mint。
-   和其他平台一样免费版本使用需要注册一个账号，当然也是免费的。
-   由于 AVID 分发限制，不包含 AVID DNxHD 授权，之后会通过官方商店为
    Linux 用户提供购买此授权的选项。
-   不支持导出 H.264 至 MOV、Quicktime/MPEG4 和 AVI 容器。
-   包含 7 天免费的 Pro 版本试用。
