Title: Mandriva Linux 2009 Spring RC 1 发布
Date: 2009-03-12 09:46
Author: toy
Category: Distros
Tags: Mandriva
Slug: mandriva-linux-2009-spring-rc-1-released

昨天，Mandriva Linux 开发团队发布了 Mandriva Linux 2009 Spring RC 1
版本，以供用户们测试。该版本除了包含 KDE 4.2.1、GNOME 2.25.92、Xfce
4.6、X.org server 1.6、OpenOffice.org 3.0.1、Qt 4.5
等更新之外，还在安装介质中加入了 HDT（Hardware Detection
Tool，硬件检测工具）和 draksnapshot 还原工具。另外，针对 USB
存储设备也有 Hybrid ISO 提供。

![Mandriva logo](http://i.linuxtoy.org/i/2007/12/mandriva-logo.jpg)

有关此版本的更多信息，可以参考其[发布公告](http://blog.mandriva.com/2009/03/11/mandriva-linux-2009-spring-rc1-ready-for-tests/)。

Mandriva Linux 2009 Spring RC 1 提供有 DVD、CD 及 Live CD 的 ISO
映像，可从其官方或镜像站点下载。

[Mandriva Linux](http://wiki.mandriva.com/en/2009.1_RC_1#Availability)
