MaDe (妈的？！) 是一款在 Chrome 中使用的所见即所得的 MarkDown 编辑器。

MaDe 提供双面板的操作方式，在左面板的键入 MarkDown
语法，右面板即可看到效果。

MaDe 由神秘的 [Shellex](http://shellex.info/) 编写。

[Chrome Web Store
地址](https://chrome.google.com/webstore/detail/oknndfeeopgpibecfjljjfanledpbkog)
