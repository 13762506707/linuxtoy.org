Title: Python 2.7 发布
Date: 2010-07-06 00:52
Author: toy
Category: News
Tags: Python
Slug: python-27

Python 团队已发布新的 2.7 版本。该版本加入了 Python 3.1  
中引入的一些新特性，包括有序字典类型、更快的 IO  
模块、新的单元测试框架（unittest）、Tkinter 封装器支持 Tile、字典  
view、sysconfig 模块等等。
