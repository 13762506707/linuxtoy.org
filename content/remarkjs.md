[Remarkjs][r] 是一个 JavaScript 库，利用它，你可以使用 Markdown  
格式来制作幻灯片。

目前，Remarkjs 提供了如下特性：

* 具有 Presenter 模式，可通过 p 键开启  
* 使用左右或上下方向键进行翻页  
* 支持语法加亮  
* 能够对幻灯进行缩放  
* 支持智能手机、平板等触屏设备  
* 因为直接在浏览器中播放，所以可以跨平台

关于 Remarkjs 的使用，只要了解一点 HTML 知识就好。具体可以看看  
Remarkjs 在 GitHub 上的 [Wiki][w] 页面。
