[Simple CSS](http://www.hostm.com/css/) 可以帮助你编写 CSS，它能够运行于
Mac OS、Windows、Linux 等系统。使用 Simple CSS 你不仅可以从零开始编写
CSS，而且也能导入存在的样式表，并重新修改它。Simple CSS
是免费软件，目前支持 CSS2 规范。
