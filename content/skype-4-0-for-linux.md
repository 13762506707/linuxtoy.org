在 M$ 收购 Skype 一年过后，Skype 4.0 for Linux 终于发布了。

相比近乎停滞的 2.2 Beta 来讲，本次代号“Four Rooms for
Improvement”的版本包含了许多变化：

-   包含全新设计的对话视图。
-   全新的通话视图，和其他平台类似。
-   大幅度改善了音频和视频通话质量。
-   新的表情图标集。

目前发布了 32 位和 64 位的 DEB 包和 32 位的 RPM 包，通用的 tar.gz
版本尚未就绪。
