Title: TeX Live 2009 发布
Date: 2009-11-11 16:10
Author: toy
Category: Apps
Tags: TeX Live
Slug: tex-live-2009

包含 TeX 及其相关程序的发行版 TeX Live 于近日发布了 2009 版本。TeX Live
2009  
主要包含以下亮点：

* LuaLaTeX 的默认输出格式为 PDF，并新增了 dviluatex 和 dvilualatex
这两个命令  
* 去掉了原来的 Omega 引擎和 Lambda 格式文件，保留更新后的 Alpeh 和  
Lamed，另外还有 Omega 工具  
* 包含新版本的 AMS Type 1 字体  
* Windows 和 MacTeX 包含了新的 GUI 前端 TeXworks  
* 在许多平台下包含了 Asymptote 图形程序  
* 单独的 dvipdfm 已被 dvipdfmx 所替代，后者包含 CJK 支持，及多年来在
dvipdfm  
基础上的修正  
* 现在包括 cygwin 和 i386-netbsd 平台下的可执行文件  
* 其他更新包括使用 xz 替代原有的 lzma  
压缩方式、在不和现有变量名冲突的情况下允许在文件中使用 $
字符、Kpathsea  
库支持多线程、整个 TeX Live 的编译现在基于 Automake 等

TeX Live 2009 可从其[官方主页](http://www.tug.org/texlive/)下载。
