Linux Man 手册的主要撰写人 Michael Kerrisk 的书 "The Linux Programming
Interface" 刚刚出版。

-   **内容详实**：全书共1552页，包含 115
    个图表，88个表格，近200个样例程序
-   **新手入门、老手参考**：循序渐进的方式讲述 Linux 内核及 glibc 的
    API，同时也方便老手查找。
-   **着重 Linux，兼顾 Unix 标准**：覆盖 i-node 标示等 Linux
    特有接口，同时对于适用于所有的 Unix 的内容参照 POSIX 标准进行组织。
