如果你像我一样每次看到长长的 Manpage 都发怵的话，那么不妨来试试 [tldr][t]。tldr 堪称 Manpage 精简版，它给你最有用的示例说明，只要依样画葫芦即可正确使用命令。

<!-- PELICAN_END_SUMMARY --> 

![tldr](/img/2014/02/tldr.png)

tldr 需要 Node，通过 npm 可以安装：

    npm install -g tldr

其源代码可从 GitHub 获取。

[t]: https://github.com/rprieto/tldr
