Title: Ubuntu 11.10 Alpha 2
Date: 2011-07-08 10:29
Author: lovenemesis
Category: Distros
Tags: Ubuntu
Slug: ubuntu-1110-alpha-2

代号 "Oneiric Oncelot" 的 Ubuntu 11.10 发布了 Alpha 2 版本，开始了 GNOME
3 整合工作。

更新的组件有：

-   Linux Kernel 3.0RC5
-   GCC 4.6.1
-   Mozilla Firefox 5.0
-   Mozilla Thunderbird 5.0
-   Mesa 7.11

最重要的是引入了 **GNOME 3.1.2** ，引入了 Deja Dup 备份软件，并且使用
[LightDM 替代了
GDM](http://linuxtoy.org/archives/lightdm-login-screen-mockup.html)。

[详细内容及下载](https://wiki.ubuntu.com/OneiricOcelot/TechnicalOverview/Alpha2)
