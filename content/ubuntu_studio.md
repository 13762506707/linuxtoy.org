[Ubuntu Studio](http://ubuntustudio.com) 是一个基于 Ubuntu
的衍生发行版，致力于为那些音视频编辑以及 CG
的爱好者提供专业水准的工具平台。这个计划目前还处在草创阶段，虽然网站已经建立但是还没有什么实际内容。对此感兴趣的朋友可以到
[Wiki](https://wiki.ubuntu.com/UbuntuStudio) 页面上去参与讨论。

喜欢尝鲜的朋友可以在自己的源里面增加下面两行：  

` deb http://archive.ubuntustudio.org/ubuntustudio edgy main deb-src http://archive.ubuntustudio.org/ubuntustudio edgy main`

然后就可以用 apt 安装 meta 包了：  

` sudo apt-get install ubuntustudio-audio ubuntustudio-plugins ubuntustudio-video ubuntustudio-graphics`

实际上这些包里的所有软件都可以在 Edgy
的源里下载到，个人觉得现在升级的价值很小。
