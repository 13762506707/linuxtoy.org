获取 Ubuntu 的版本信息：

-   cat /etc/issue－获悉 Ubuntu 的版本号
-   uname -r－获知所用的 Linux 内核版本

检查性能状态：

-   cat /proc/meminfo－提供内存占用的信息

使用 root 模式：

-   sudo -s -H－作为 root 用户切换到 root 模式
-   sudo -s－切换到 root 模式，但仍为当前用户
