Title: Vim 7.3b 放出
Date: 2010-07-27 18:28
Author: toy
Category: Apps
Tags: Vim
Slug: vim-73b

Bram Moolenaar 于昨日放出了 Vim 7.3 的 Beta
版本，以供其用户测试之用。该版本除增添了一些新的特性之外，还修正了之前版本中的不少缺陷。

Vim 7.3 的主要变化包括：

+ 支持持久性撤销  
+ Blowfish 加密  
+ Conceal text  
+ Lua 接口  
+ Python 3 接口

Vim 7.3b 可从[这里下载](ftp://ftp.vim.org/pub/vim/unstable/)。
