M$ 的 Visual Studio 家族新添一名支持 Linux
的成员，成为它家首个跨平台的代码编辑器。

<!-- PELICAN_END_SUMMARY -->

Visual Studio Code 是一款面向云端应用开发的代码编辑器（轻量级
IDE？），现阶段具备的功能有：

* 具备家族标志性的代码补全功能  
* 支持 Node.js 和 ASP.NET v5 应用开发  
* 具备 Node.js 调试功能，未来还将增添更多支持  
* 内置 Git 支持

[官方首页及多平台下载](https://code.visualstudio.com/)
