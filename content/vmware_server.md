[VMware](http://www.vmware.com) 推出了旗下 VMware Server 产品的 1.0
版，支持 SUSE Linux 10.1、Ubuntu 6.x、Sun Solaris 10.x、FreeBSD 6.0 等
32 位和 64 位操作系统，新增几项关键特性，提供了对于其他 VMware
产品的支持，同时也包括 VMware API 支持。
