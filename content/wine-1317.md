开源 Win16/32/64 API 实现 Wine 发布 1.3.17 版本，带来 MSI
安装补丁初步支持。

-   实现新的 **Vista 文件对话框**。
-   初步支持以 **MSI 安装器方式的补丁安装**。
-   提升日历控制组件。
-   修复文件关联相关的 Bug。
-   清理了内置程序的菜单项。
-   各种 Bug 修复。
