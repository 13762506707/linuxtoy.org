[XPontus](http://xpontus.sourceforge.net) 是基于 Java 的 XML
编辑器，主要特性有：XML 验证、语法高亮显示、代码缩进、XSL 转换、多种
XSLT 处理器、打开远程文件、Schema
验证、多语言支持、大多数文本编辑功能、多平台支持等。
